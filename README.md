# Loly Hair Salon Website

## Mô tả dự án
Website giới thiệu dịch vụ salon và hỗ trợ khách hàng đặt lịch hẹn trực tuyến.
Dự án được thực hiện nhằm hiểu cách một hệ thống web cơ bản vận hành, từ giao diện người dùng đến xử lý dữ liệu.

## Mục tiêu dự án
- Hiểu luồng xử lý của một hệ thống đặt lịch hẹn
- Thực hành phân tích yêu cầu và chức năng hệ thống
- Làm việc với dữ liệu người dùng và giao diện web
- Rèn luyện tư duy logic và làm việc theo quy trình

## Chức năng chính
- Hiển thị thông tin và dịch vụ salon
- Cho phép khách hàng đặt lịch hẹn
- Nhập và lưu trữ thông tin khách hàng
- Kiểm tra tính hợp lệ của dữ liệu đầu vào

## Công nghệ sử dụng
- HTML
- CSS
- JavaScript
  
## Ghi chú
Dự án được sử dụng cho mục đích học tập và thực hành, tập trung vào việc hiểu hệ thống, quy trình xử lý và trải nghiệm người dùng.
